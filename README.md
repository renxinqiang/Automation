### 从ZOL首页开始,给文章添加一条评论

#### 配置
* selenium
* 火狐浏览器(无痕)
* 延时处理

#### 流程

1. 启动火狐浏览器
2. 打开ZOL首页
3. 选中搜索栏新闻
4. 输入"iphone X"
5. 点击搜索,跳转搜索页面
6. 选中新标签页
7. 清空输入框
8. 输入"iphone 8"
9. 点击搜索
10. 选中页面列表第6个链接
11. 点击链接,跳转文章页面
12. 选中新标签页
13. 模拟点击导航栏登录按钮
14. 选中登录iframe
15. 输入用户名密码
16. 点击登录
17. 刷新页面使登录信息加载到评论区
18. 滚动到评论区域
19. 选中评论框并清空评论框
20. 输入评论并点击评论
